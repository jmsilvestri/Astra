# Project History

Astra was created by Johanna as an open source project to build a local AI voice assistant that feels more capable, more private, and more personal than mainstream cloud assistants.

The initial goal is to create a voice assistant that can eventually support natural conversation, local reasoning, useful personal integrations, and a delightful home experience without depending on paid token APIs at runtime.

## Founding Decisions

The early direction for Astra includes:

- Use Ollama for local LLM reasoning.
- Build local-first wherever practical.
- Keep the primary runtime free from per-token costs.
- Preserve privacy as a core product constraint.
- Organize the system as modular, replaceable components.
- Develop the project openly with professional documentation.

Sprint 0 establishes the shared foundation before business logic, voice features, tool calling, memory, or integrations are implemented.
